1、下载插件
官网下载
http://xmlstar.sourceforge.net/download.php
下载链接备份
https://nchc.dl.sourceforge.net/project/xmlstar/xmlstarlet/1.6.1/xmlstarlet-1.6.1-win32.zip

2、解压后将xml.exe复制到EditPlus安装根目录
注意：只需要复制xml.exe单个文件即可

3、配置EditPlus
Tools=> configure user tools =>add toolprogram
配置参数：
1.menu text:菜单名称，可随意配置，xml formatter
2.command:配置xml.exe文件路径
3.argument：fo
4.action:run as text filter (Replace)
